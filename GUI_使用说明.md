# 有道云笔记GUI管理工具使用说明

## 🚀 启动应用

```bash
python youdao_gui.py
```

## 📋 功能介绍

### 1. 主界面功能

- **文件列表**: 显示当前目录下的所有文件和文件夹
- **路径显示**: 显示当前所在的路径位置
- **工具栏**: 包含各种操作按钮

### 2. 工具栏功能

- **刷新**: 重新加载当前目录内容
- **返回上级**: 返回到根目录（简化实现）
- **搜索框**: 输入关键词搜索文件和文件夹
- **搜索按钮**: 执行搜索操作
- **批量下载**: 下载选中的多个项目
- **选择下载目录**: 设置文件下载保存位置

### 3. 文件列表操作

#### 双击操作
- **双击文件夹**: 进入该文件夹
- **双击文件**: 无操作

#### 右键菜单
- **下载**: 下载选中的文件或文件夹
- **进入文件夹**: 进入选中的文件夹（仅对文件夹有效）
- **复制ID**: 复制文件/文件夹的唯一ID到剪贴板
- **复制路径**: 复制文件/文件夹的完整路径到剪贴板

### 4. 搜索功能

1. 在搜索框中输入关键词
2. 按回车键或点击"搜索"按钮
3. 搜索结果会显示匹配的文件和文件夹
4. 搜索结果显示完整路径，方便定位

### 5. 下载功能

#### 单个下载
1. 右键点击要下载的文件或文件夹
2. 选择"下载"
3. 等待下载完成

#### 批量下载
1. 按住Ctrl键选择多个文件/文件夹
2. 点击工具栏的"批量下载"按钮
3. 确认下载操作
4. 查看下载进度

### 6. 下载目录设置

1. 点击"选择下载目录"按钮
2. 在弹出的对话框中选择保存位置
3. 所有下载的文件将保存到该目录

## 📊 状态信息

### 状态栏显示
- **当前操作状态**: 显示正在执行的操作
- **进度条**: 显示下载进度
- **下载目录**: 显示当前设置的下载保存位置

### 文件信息显示
- **名称**: 文件或文件夹名称
- **类型**: 文件或文件夹
- **大小**: 文件大小（文件夹显示为"-"）
- **修改时间**: 最后修改时间
- **ID**: 有道云笔记中的唯一标识符

## 🔧 技术特性

### 自动登录
- 应用启动时自动使用cookies.json中的认证信息登录
- 登录成功后自动加载根目录内容

### 后台操作
- 搜索和下载操作在后台线程中执行
- 不会阻塞界面响应

### 错误处理
- 网络错误和API错误会显示友好的错误消息
- 操作失败时会在状态栏显示错误信息

## 📝 使用技巧

1. **快速搜索**: 在搜索框中输入关键词后直接按回车键
2. **批量操作**: 使用Ctrl+点击选择多个项目进行批量下载
3. **路径复制**: 使用右键菜单的"复制路径"功能快速获取文件路径
4. **ID复制**: 复制文件ID可用于其他脚本工具

## ⚠️ 注意事项

1. **网络连接**: 确保网络连接正常，操作需要访问有道云笔记服务器
2. **Cookie有效性**: 如果登录失败，请检查cookies.json文件是否有效
3. **下载目录**: 确保下载目录有足够的磁盘空间
4. **大文件下载**: 下载大文件或大文件夹时请耐心等待

## 🐛 故障排除

### 登录失败
- 检查cookies.json文件是否存在且有效
- 重新提取浏览器中的cookie信息

### 下载失败
- 检查网络连接
- 确认下载目录有写入权限
- 检查磁盘空间是否充足

### 界面无响应
- 等待后台操作完成
- 如果长时间无响应，重启应用

## 📞 支持

如果遇到问题，请检查：
1. cookies.json文件是否正确配置
2. 网络连接是否正常
3. 有道云笔记账户是否正常
